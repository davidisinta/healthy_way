Welcome to Zeraki's Chatbot API

Introduction

This API provides access to Zeraki's powerful chatbot. It is knowledgeable on Zeraki's services. You can integrate it into your applications to enable natural language interactions with users.

Usage

To use this API, you need to make HTTP requests to the following endpoints:

  • POST /v2/chat - Send a message to the chatbot and receive a response.

Example

Here is an example of how to interact with the chatbot endpoint:


curl -X POST "https://zerakicopilotapp.azurewebsites.net//v2/chat" \
     -H "Content-Type: application/json" \
     -d '{
           "text": "hello there!!"
         }'

Response:


                {
                    "documents":[],
                    "reply":"Hello! How can I assist you today?"
                }
